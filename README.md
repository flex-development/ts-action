## Getting Started

TypeScript GitHub Action template. To get started:

1. Set your organization and project name. Perform a search and replace for:

   - `flex-development` to set your organization
   - `ts-action` to set the project name

2. Update [`package.json`](package.json) details
3. Update [action metadata](action.yml)
4. Update [commit scopes and types](commitlint.config.js)
5. Update [repository labels](.github/labels.yml)
6. Update [workflow](.github/workflows) environment variables and secrets
7. Update the project [pull request template](.github/PULL_REQUEST_TEMPLATE.md)
   and [issue templates](.github/ISSUE_TEMPLATE)
8. Update project [Contributing Guide](CONTRIBUTING.md)
9. Update project [license](LICENSE.md)
10. Update project `README` with your project details
